Frequently Asked Questions

What is React ?

React is a JavaScript library for building user interfaces. React enables developers to create reusable UI components and to efficiently update the user interface in response to changes in data.

What is JSX ?

JSX is a syntax extension for JavaScript used in React to make it easier to write and manipulate HTML-like code within JavaScript.

What is a component ?

A component in React is a modular, reusable piece of code that represents a part of the user interface.
